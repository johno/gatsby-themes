# Flex

`Flex` is used as the wrapping element for a flexbox layout.
It sets `display: flex;` and exposes a prop API for aligning and justifying its children.

```js react-live
<Flex
  flexWrap='wrap'
  alignItems='center'
  justifyContent='space-between'
  children={items.map(item =>
    <Box key={item.id} w={[1/2 1/3 1/4]}>
      <Item {...item} />
    </Box>
  )}
/>
```
