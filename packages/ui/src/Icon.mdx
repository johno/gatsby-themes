# Icon

The gatsby-ui design system supports the following icons

```js react-live
<Flex justifyContent="space-between" flexWrap="wrap">
  <Icon name="github" />
  <Icon name="spectrum" />
  <Icon name="twitter" />
  <Icon name="chevronDown" />
  <Icon name="chevronLeft" />
  <Icon name="chevronRight" />
  <Icon name="chevronUp" />
  <Icon name="warning" />
  <Icon name="info" />
</Flex>
```
