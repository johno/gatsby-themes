# Icon

Currently, the unified design system supports the following icons

```.jsx
<Flex justifyContent="space-between" flexWrap="wrap">
  <Icon name="github" />
  <Icon name="spectrum" />
  <Icon name="twitter" />
  <Icon name="chevronDown" />
  <Icon name="chevronLeft" />
  <Icon name="chevronRight" />
  <Icon name="chevronUp" />
  <Icon name="warning" />
  <Icon name="info" />
</Flex>
```
